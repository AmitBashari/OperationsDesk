# Workstation Prototype

A passive gem income system prototype for Raid: Shadow Legends concept.

## Features

- **2 Investment Proposals**: Growth (24h) and Commitment (7d)
- **Investment Slider**: Choose how much to invest (10-1500 gems based on level)
- **Workstation Levels**: Upgrade to unlock higher investment caps
- **Real-time Countdown**: Track your active proposals
- **Mobile Optimized**: Works seamlessly on desktop and mobile devices

## How to Use

1. Select a proposal (Growth or Commitment)
2. Adjust your investment amount using the slider
3. See your potential return range
4. Start the proposal and wait for completion
5. Collect your rewards!

## Design Principles

- **Predictable**: No RNG loss, only bounded variance on profit
- **Cancel-friendly**: Get your investment back anytime
- **Transparent**: See exactly what you'll get back
- **Dignified**: Player-driven choices over passive dripping
